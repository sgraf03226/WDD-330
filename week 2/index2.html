Javascript: Novice to Ninja

Chapter 2

~ Primitive data types include strings, numbers, bigints, booleans, undefined, and symbols.

~ Non-primitive data types are objects, arrays, functions, and object laterals.

~ The let keyword declares variables in black scope. Block scope means that the variables only exist in that particular block.

~ Template literals are useful for interpolation, which allows you to embed an expression within a string. The template literal is a string within backticks and an $() is the place holder.
