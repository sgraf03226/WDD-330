HTML5 and CSS3 for the Real World

Chapter 8

~ Transforms allow you to rotate, scale or skew elements. Unfortunately transforms do not work on inline elements and you will need to add display: inline-block.

~ Translation functions allow the movement of elements to the left, right, up, or down.

~ Transitions will allow the values of CSS properties to change over time (simple animations). CSS animations however, allow control over each step of the animation using keyframes. A keyframe is a snapshot that defines the start or endpoint of smooth transitions.

Chapter 12

~ To create a canvas element, you must create the element in the element declaration and proide it with an ID, width, and height.

~ Different from canvas, SVG images are readily available through the DOM. Becuase SVG's are complex, an image editor (Inkscape) is needed to to create an image.

~ Event listeners are used for drag and drop options.
