Chapter 5

~ Most of this chapter was review..

~ JS objects are collections of properties, properties are associations between a name and a value.

~ If a properties value is a function, then it is called a method.

~ Object literals in JS are a list of name-value pairs seperaated by commas and wrapped in curly braces. The purpose of an object literal is to encapsulate data and minimize the use of global variables.

~ Date object properties require getter and setter methods.

Chapter 6

~ Node lists are similar to arrays but do not use the array methods (splice, slice and join). They can be converted to arrays using the spread operator or Array.form.

~ Classes in JS can be set using setAttribute() method or using the className property. If there are multiple classes, the className property will overwrite those existing classes. Instead, to add a class use classList.add(). You can then turn a class on or off using classList. toggle().
