Javascript: Novice to Ninja

Chapter 9

~ You can create global variable without using const, let, or var. It is good to note that you should avoid referencing these global variables using window (window.globalVariable). The reasoning behind this is so that the code is portable between environments.

~ Confirm() is always a boolean and is dependent on the user's response. Alert() should always return undefined. Prompt() is a string that is dependent on what the user has entered.

~ Review window methods.

Chapter 14

~ To embedd data in a web page using custom attributes that are ignored by the browser, use Javascript and data-attribute. The .getAttribute accesses the data-attributes.

~ There are different forms of API, the geoLocation API provides the location and speed of a device, where the web storage API provides local storage functionality. To show messages using the system's notifications, use notification API. There is also a canvas API.

~ The protocol websockets allows two-way communication with a server. The server pushes responses to the client over an open connection that stays open.
